# Proposal: Oh My Professors (澳洲大学点评平台)

## 1. Executive Summary
"Oh My Professors" 是一个专为澳洲留学生和本地大学生设计的垂直点评与推荐平台。通过整合大学、教授、课程代码 (Course Code) 的多维评价数据，解决选课阶段的信息不对称问题，帮助学生规避“雷课”，寻找“水课”与名师。

## 2. Market Problem
- **信息碎片化**：目前学生获取教授和课程评价主要依靠社交媒体 (如 Reddit, 小红书) 或碎片的群聊，缺乏系统性的整合。
- **澳洲本土化不足**：Rate My Professors 虽然知名度高，但在澳洲大学的覆盖率和数据更新频率有限。
- **选课维度单一**：现有平台 (如 StudentVIP) 侧重于二手书和笔记，对“教授教学风格”和“课程给分好坏”的深度挖掘不足。

## 3. Core Solution
- **教授评价系统**：支持针对特定教授的评分、点评、标签 (如 "Easy A", "Heavy Reading")。
- **课程代码搜索**：通过 Course Code (如 COMP1000) 直接链接至关联教授及历史评价。
- **智能推荐系统**：基于历史数据推荐“高分水课”或“学术名师”。
- **社区闭环**：结合社交属性，允许学生分享选课心得。

## 4. Key Features (MVP)
- 大学/教授/课程 基础搜索与展示。
- 点评发布系统 (支持匿名)。
- 排行榜 (Top Rated Professors, Easiest Courses)。
- 澳洲主流大学元数据集成。

## 5. Monetization (Draft)
- 留学生服务机构广告。
- 职业培训、求职辅导精准推广。
- 补习/笔记交易引流。

## 6. Target Audience
- 澳洲所有大学的在读留学生（尤其是中国留学生）。
- 澳洲本地大学生。
