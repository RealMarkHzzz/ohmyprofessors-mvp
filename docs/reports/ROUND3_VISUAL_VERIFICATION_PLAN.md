## 验收标准

### 必须达到
- ✅ 颜色色值完全匹配 (±2 RGB)
- ✅ 字体大小误差 ±2px
- ✅ 间距误差 ±2px
- ✅ 圆角值完全匹配
- ✅ 阴影效果视觉一致

### 可接受差异
- ⚠️ 内容文字不同 (正常)
- ⚠️ 图标 SVG 路径略有不同 (视觉效果一致即可)
- ⚠️ 动画时长 ±50ms

### 不可接受差异
- ❌ 主色调错误
- ❌ 布局比例明显不同
- ❌ 响应式断点不一致
- ❌ Hover 效果缺失

## 执行步骤

### Step 1: 准备环境
```bash
# 等待 Vercel 部署完成
# 检查部署状态
curl -I https://ohmyprofessors-mvp.vercel.app
```

### Step 2: 截图对比
1. 打开两个浏览器窗口并排
2. 左侧打开参考模板
3. 右侧打开实施网站
4. 逐个 Section 对比截图

### Step 3: 像素级测量
使用浏览器开发工具测量关键元素:
```
- 导航栏高度
- 容器最大宽度
- 按钮内边距
- 卡片圆角
- 字体大小
```

### Step 4: 交互测试
1. Hover 所有按钮
2. Hover 所有卡片
3. 展开移动菜单
4. 切换 Dark Mode

### Step 5: 记录结果
在 `ROUND3_VISUAL_QA_RESULTS.md` 中记录:
- ✅ 通过的检查项
- ⚠️ 需要调整的项目
- ❌ 失败的项目
